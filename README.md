# HR Employee Attrition Analysis 📊

## 📌 Project Overview

Employee attrition is a major challenge for organizations. High attrition rates can lead to increased recruitment costs, loss of skilled employees, and reduced productivity.

This project analyzes HR employee data to identify the key factors that influence employee attrition and helps organizations make better HR decisions.

---

## 🎯 Objective

The main objectives of this project are:

* Analyze employee data to understand attrition patterns
* Identify factors affecting employee turnover
* Provide insights to improve employee retention
* Support data-driven HR decision making

---

## 📂 Dataset

The dataset used in this project is the **IBM HR Employee Attrition Dataset**.

It contains information about employees such as:

* Age
* Job Role
* Salary
* Department
* Work Experience
* Job Satisfaction
* Attrition Status

---

## 🛠 Tools & Technologies Used

* Python 🐍
* Excel 📑
* Power BI 📊

* Data Analysis & Visualization

---

## 🔎 Analysis Performed

The following analysis was performed:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Attrition Rate Analysis
* Department-wise Attrition
* Salary vs Attrition Analysis
* Job Role Analysis
* Employee Satisfaction Analysis

---

## 📈 Key Insights

Some important insights from the analysis:

* Employees with low job satisfaction have higher attrition rates
* Certain departments show higher employee turnover
* Salary and work experience impact attrition
* Work environment plays an important role in employee retention
